Implement modules/reporter.py and finalize main.py for DefenseScan:

Reporter requirements:
- Console summary with color coding by risk level
  (Critical=RED, High=ORANGE, Medium=YELLOW, Low=CYAN, Info=WHITE)
- Finding count grouped by risk level
- Top 5 highest risk findings display
- JSON report with full details + scan metadata

main.py requirements:
- argparse CLI with options:
    --web <URL>       web scan target
    --system          run system scan
    --output <file>   JSON output path (default: report_YYYYMMDD.json)
    --threads <n>     concurrent threads (default: 5)
    --timeout <n>     request timeout seconds (default: 10)
    --verbose         detailed output
- Admin privilege check on startup for system scan
- Windows version detection and display
- Combined report when both --web and --system used
