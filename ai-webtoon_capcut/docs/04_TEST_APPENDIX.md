# 테스트 계획 보충: Suno 자막 정규화

## 단위 테스트

- 대괄호 섹션 태그 분류
- 연주·음향 지시 분류
- 실제 발화 후보 분류
- cue 밀도와 중앙 지속 시간 계산
- 반복 가사 시퀀스 탐지
- 장기 cue 이상 탐지
- 동일 타임스탬프 우선순위
- 조기 End marker 처리
- 마지막 가사 이후 instrumental outro 계산

## 회귀 샘플 구성

최소 다음 유형의 Suno 곡을 포함한다.

1. 정상 가사만 있는 LRC/SRT
2. 앞부분에 전체 프롬프트가 압축 삽입된 곡
3. Guitar Solo 같은 연주 태그가 있는 곡
4. 드럼 카운트·함성·호흡이 있는 곡
5. Remix 또는 Extend로 구조가 달라진 곡
6. 같은 후렴을 반복한 곡
7. 일부 가사를 생략한 곡
8. 긴 instrumental outro가 있는 곡

## 속성 테스트

- 정리 후 cue는 음원 길이를 벗어나지 않는다.
- `lyric`이 아닌 cue는 기본 SRT에 포함되지 않는다.
- 모든 제거 항목은 metadata 또는 review 파일에 남는다.
- 동일 입력은 동일한 분류 결과를 만든다.
- 원본 파일 해시는 처리 전후 동일하다.
