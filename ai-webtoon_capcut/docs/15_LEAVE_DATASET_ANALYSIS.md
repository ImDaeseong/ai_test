# 떠나고 싶어 데이터셋 분석

## 1. 판정

- 검증 곡 순번: 3
- 데이터 상태: `BUILD_READY_WITH_WARNINGS`
- 영상 생성 가능 여부: 가능
- 자동 렌더링 투입 여부: 규칙 기반 정규화 후 가능
- 원본 경로: `ai-webtoon/output/떠나고 싶어`

`떠나고 싶어`에는 이미지, 패널 문서, WAV, LRC, SRT와 원문 가사가 모두
존재한다. 이미지 번호와 패널 문서도 완전히 대응한다. `디저트`에서 발견된
44초 장시간 자막과 같은 심각한 시간축 오류는 없다.

대신 느린 확대·이동 또는 한 컷 내부의 모션 변형을 적용한다.

## 5. 섹션 정규화

파일명은 일부 구간을 축약한다.

- `panel_003`-`panel_006`: 파일명 `verse`, 내부 섹션 `Verse 1`
- `panel_016`-`panel_019`: 파일명 `verse`, 내부 섹션 `Verse 2`
- `panel_023`-`panel_028`: 파일명 `chorus`, 내부 섹션 `Final Chorus`

이는 이미지가 잘못된 것이 아니라 파일명 토큰의 정보 손실이다. 다음
우선순위로 정규화한다.

1. 패널 문서 내부 `섹션`
2. `01_storyboard.md` 테이블
3. 파일명 섹션 토큰

`verse`를 임의로 하나의 구간으로 합치거나 `Final Chorus`를 일반
`Chorus`로 축소하면 안 된다.

## 6. LRC/SRT 검사

SRT는 총 59개 큐다.

- 실제 가사 큐: 40개
- Suno 섹션/연주 지시문: 19개
- 첫 큐 시작: 17.123초
- 마지막 가사 종료: 175.054초
- 가사 큐 중앙값: 4.189초
- 가장 긴 가사 큐: 5.904초
- 10초 초과 가사 큐: 없음
- 중첩 쌍: 13개

중첩은 같은 시점의 섹션 태그, 악기 지시문, 화음 지시문과 실제 가사에서
발생한다. 대괄호 지시문은 화면에서 제거하고 분석 메타데이터로 보존한다.
가사 시간은 전체적으로 정상 범위여서 WhisperX 재정렬은 필수가 아니라
선택적 품질 검사로 적용할 수 있다.

### 누락된 Outro 태그

원문 가사에는 다음 구간이 있다.

`[Outro: Instrumental Only, fading guitar loop, ambient night texture]`

하지만 다운로드 LRC/SRT에는 Outro 태그가 없다. 마지막 가사는
175.054초에 끝나고 음원은 193.152초에 끝나므로 남은 18.098초를
`Outro`로 생성한다. 이 구간에는 `panel_029`와 `panel_030`을 배치하고
화면 자막은 표시하지 않는다.

## 7. 권장 타임라인 처리

1. 0-17.123초: Intro 이미지 2장
2. 17.123-52.787초: Verse 1 이미지 4장
3. 52.787-70.340초: Pre-Chorus 이미지 3장
4. 70.340-104.411초: Chorus 이미지 6장
5. 104.411-122.362초: Verse 2 이미지 4장
6. 122.362-140.314초: Bridge 이미지 3장
7. 140.314-175.054초: Final Chorus 이미지 6장
8. 175.054-193.152초: Outro 이미지 2장

위 경계는 다운로드 자막의 섹션 시작점과 음원 종료점을 이용한 1차
타임라인이다. 프레임 변환 시 마지막 프레임을 음원 종료에 고정한다.
8초를 넘는 Intro, Verse 1, Outro 컷에는 정적인 장면 반복보다 Ken Burns
이동과 완만한 줌을 적용한다.

## 8. 프로그램 검증 항목

이 곡은 다음 기능을 검증하는 세 번째 실제 사례다.

- 공백이 포함된 한글 폴더와 파일명 처리
- 30장 이미지로 약 3분 음원 구성
- 17초 이상의 전주와 후주 처리
- 누락된 Outro 섹션 복원
- `Verse 1/2`, `Final Chorus` 세부 섹션 보존
- 축약 파일명보다 내부 메타데이터 우선 적용
- 소규모 혼합 해상도 자동 정규화
- 정상 자막에서 불필요한 WhisperX 재처리 회피

## 9. 통과 조건

- 30개 이미지가 누락 없이 한 번 이상 배치된다.
- 최종 영상 길이가 193.152초와 프레임 오차 이내로 일치한다.
- 17.123초 전주와 18.098초 후주에서 빈 화면이 생기지 않는다.
- 대괄호 지시문이 화면 자막으로 출력되지 않는다.
- Verse 1과 Verse 2가 서로 다른 섹션으로 유지된다.
- Final Chorus가 일반 Chorus로 잘못 합쳐지지 않는다.
- Outro 두 이미지가 마지막 무자막 구간에 배치된다.
- 혼합 해상도 이미지에 왜곡이나 검은 틈이 발생하지 않는다.

## 10. 결론

`떠나고 싶어`는 현재 세 곡 중 자막 시간축이 가장 안정적인 데이터다.
규칙 기반 정규화만으로 자동 영상 생성에 투입할 수 있으며, WhisperX는
선택적 검수 단계로 두는 것이 적합하다. 이미지 수가 30장으로 적지만 평균
노출 시간이 6.438초이므로 반복 없이 전체 음원을 자연스럽게 채울 수 있다.
