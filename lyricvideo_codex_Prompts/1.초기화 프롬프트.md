- Loads local MP3 audio

# REMOTION IMPLEMENTATION REQUIREMENTS

## Media Handling

- Store all media under:
  `public/media/`

- Use:
  `staticFile()`

- Use:
  `Audio` from `remotion`

---

## Lyric Timing

### LRC Priority

Parse:
- `[mm:ss.xx]`
- Multiple timestamps per line
- UTF-8 text

### SRT Fallback

Support:
- hour/minute/second/millisecond timing
- multiline subtitle merging

---

## Rendering Logic

Compute active lyric line using:

- `useCurrentFrame()`
- `fps`

Do NOT use timers or intervals.

---

## Visual Behavior

Display:

- Previous line
- Current line
- Next line

Current line must:
- scale slightly larger
- brighter/highlighted
- animated smoothly

Neighbor lines must:
- fade
- blur slightly
- reduce opacity

Avoid dense subtitle-wall rendering.

---

## Styling Requirements

Must support:

- Korean
- Japanese
- English
- mixed UTF-8 lyrics

Use responsive typography.

Prevent:
- text clipping
- overflow
- layout jumping

---

## Duration Rules

If audio duration probing exists:
- use actual duration

Otherwise:
- estimate using latest lyric timestamp + 3~5 seconds outro

Never hardcode duration.
