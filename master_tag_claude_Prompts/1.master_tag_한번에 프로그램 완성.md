🚀 Claude 전용 프롬프트: Suno AI 전문 마스터링 엔지니어링 시스템
작업 지시: Suno AI 음원 전용 로컬 자동 마스터링 프로세서 (Windows용)

지금부터 현재 디렉토리에서 작업을 수행하라. 설명은 생략하고 데이터 엔지니어링 관점의 코드 작성 및 실행 준비를 완료하라.

🚨 절대 규칙 (반드시 준수)

❌ 외부 유료 API(OpenAI, Landr 등) 사용 금지

✅ **로컬 Python 라이브러리(Pedalboard, Librosa, Pyloudnorm)**만 사용

✅ Suno AI 특유의 결함(고음역대 손실, 저음역대 빈약함) 보정에 초점

✅ Windows 10/11 환경 최적화 및 CPU 환경 실행 가능

🛠️ 프로젝트 구조 및 파일 생성

main.py: 마스터링 핵심 로직 및 CLI 인터페이스

requirements.txt: 필수 라이브러리 명시

🎯 최종 목표 (엔지니어링 파이프라인)

데이터 분석: Librosa를 통한 주파수 응답(FFT) 및 Peak/RMS 분석

헤드룸 확보: Gain Staging을 통해 입력 신호를 -6dB로 조정

DSP 체인 (Pedalboard):

High-Pass Filter: 30Hz 이하 초저역 노이즈 제거

Parametric EQ: Suno 특유의 답답한 대역(250-500Hz) 정리 및 고역대(10kHz+) 샤프니스 보정

Compressor: 소리의 밀도와 레이어링 강화

Limiter: 피크 방지 및 최종 음압 확보

Loudness Normalization: Pyloudnorm을 사용하여 **-14 LUFS(유튜브 표준)**로 강제 고정

🔧 main.py 구현 지침

입력 방식: sys.argv[1]로 파일 경로를 받거나, 없으면 input()으로 입력받음

오디오 로드/저장: Pedalboard의 AudioFile을 사용하여 고음질(WAV/FLAC) 유지

진행 로그: 단계별(분석 중 -> 이펙트 적용 중 -> LUFS 정규화 중 -> 완료) 한글 로그 출력

예외 처리: 파일 미존재, 오디오 포맷 오류, 라이브러리 미설치 시 친절한 안내 메시지 출력

💡 코드 품질 요구사항

함수형 구조로 설계하여 추후 Go 스케줄러와 연동 가능하게 작성

시니어 개발자가 커스터마이징하기 좋게 EQ 및 Compressor 파라미터에 상세 한글 주석 포함
