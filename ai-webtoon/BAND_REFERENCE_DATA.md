# Band Performance Reference Data

## Data Policy

이 데이터는 실존 밴드를 이미지로 복제하기 위한 자료가 아니다. 공개된 공식 자료에서 공연 제작의 일반 요소를 추출하고, 오리지널 스켈레톤 밴드에 적용 가능한 속성으로 변환한다.

추출 대상:

- 무대 배치
- 조명 변화
- 카메라 거리와 방향
- 멤버 동선과 합
- 관객을 화면에 넣는 방식
- 곡별 의상 변형 범위

제외 대상:

- 얼굴과 신체 특징
- 로고와 앨범 아트
- 정확한 의상 복제
- 고유 소품과 시그니처 무대의 동일 재현
- 프롬프트 내 실존 아티스트명

## Maintenance

새 자료를 추가할 때는 공식 URL, 관찰 내용, 변환된 속성, 금지할 복제 요소를 함께 기록한다. 프로필은 특정 곡 제목이 아니라 재사용 가능한 장르·BPM·감정 규칙으로 연결한다.
