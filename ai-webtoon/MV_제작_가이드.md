# ai-webtoon MV 제작 가이드

> 패널 프롬프트 생성 → 이미지 생성 → CapCut 편집 → YouTube 업로드까지 전체 흐름

---

## 전체 흐름 요약

```
[ai-webtoon 자동 생성]
output/[곡명]/panels/panel_NNN_*.md  (프롬프트 파일)
         ↓
[이미지 생성 AI]
GPT Image / Nijijourney / FLUX.1 / Gemini
         ↓
[이미지 저장]
image/[곡명]/panel_001.png ~ panel_NNN.png
         ↓
[CapCut 편집]
이미지 나열 → Ken Burns → 음악 삽입 → 자막 → 내보내기
         ↓
[YouTube 업로드]
```

---

## STEP 1 — 패널 프롬프트 확인

### 웹 뷰어 사용 (권장)
```
실행_web.bat 실행
→ http://127.0.0.1:5300
→ 곡 선택 → 패널 카드 클릭
→ GPT / Nijijourney / FLUX.1 / Gemini 탭 선택
→ [프롬프트 복사] 버튼
```

### 파일 직접 확인
```
output/[곡명]/01_storyboard.md    ← 전체 순서 확인
output/[곡명]/panels/panel_001_*.md  ← 개별 패널 프롬프트
```

**작업 전 반드시 확인:**
- `output/[곡명]/00_style_reference.md` — 스타일 기준 블록
- `ai_img_video_prompt/reference/` — 레퍼런스 이미지 (반드시 첨부)

## STEP 2 — 이미지 생성

### 이미지 도구 선택 기준

| 도구 | 비용 | 품질 | 특징 | 권장 용도 |
|------|------|------|------|----------|
| **GPT Image (gpt-image-2)** | 유료 | 최상 | 레퍼런스 이미지 인식 최고 | 캐릭터 고정 필수 패널 |
| **Nijijourney (--niji 7)** | 저비용 | 상 | 애니/웹툰 스타일 최적 | 스타일 완성도 중요 패널 |
| **FLUX.1** | 무료 | 중상 | 자연어 문장 방식 | 배경/분위기 패널 |
| **Gemini / Imagen 3** | 무료 | 중 | 한국어 가능 | 빠른 초안 확인 |

---

### GPT Image (ChatGPT) 사용법

1. ChatGPT → 이미지 생성 모드 열기
2. **레퍼런스 이미지 첨부** (매번 필수)
   - `ai_img_video_prompt/reference/` 폴더의 PNG 파일 1~2장 첨부
3. 프롬프트 붙여넣기 (웹 뷰어 복사 버튼 사용)
4. 생성 후 저장

```
저장 위치: image/[곡명]/panel_001.png
           image/[곡명]/panel_002.png
           ...
```

> **주의:** 레퍼런스 이미지 없이 생성하면 캐릭터가 달라짐. 반드시 매번 첨부.

---

### Nijijourney (Discord) 사용법

1. Discord → Nijijourney 봇 채널
2. `/imagine` 명령어 입력
3. 프롬프트 붙여넣기 (Nijijourney 탭의 내용 — 이미 `--niji 7 --ar 16:9 --s NNN` 포함)
4. 생성된 4장 중 가장 좋은 것 선택 → `U1~U4` 버튼으로 업스케일
5. 다운로드 → 저장

```
추천 파라미터:
  발라드/감성: --niji 7 --ar 16:9 --s 150
  팝/일반:    --niji 7 --ar 16:9 --s 250
  액션/록:    --niji 7 --ar 16:9 --s 400 --style expressive
```

> **주의:** Nijijourney는 레퍼런스 이미지 첨부 불가. 캐릭터 고정은 프롬프트 텍스트에만 의존.

---

### FLUX.1 (무료) 사용법

FLUX.1을 사용할 수 있는 무료 사이트 중 하나 이용:
- Hugging Face Spaces (flux 검색)
- Fal.ai (일정 무료 크레딧)
- Replicate (일정 무료 크레딧)

1. FLUX.1 탭의 프롬프트 복사
2. 해당 사이트 접속 → 프롬프트 붙여넣기
3. 이미지 크기: `1792×1024` (16:9) 또는 `1024×1024`
4. 생성 후 다운로드

## STEP 4 — CapCut 편집 (핵심)

### 4-1. 새 프로젝트 만들기

1. CapCut 실행
2. **새 프로젝트** 탭
3. 화면 비율 선택:
   - YouTube MV → **16:9** (1920×1080)
   - YouTube Shorts → 9:16 (1080×1920)
4. 프로젝트명: 곡 제목 입력

---

### 4-2. 이미지 불러오기 및 순서 배열

1. 하단 메뉴 → **미디어** → **가져오기**
2. `image/[곡명]/` 폴더에서 이미지 전체 선택
3. **스토리보드(`01_storyboard.md`) 순서대로** 타임라인에 배열
   - panel_001 → panel_002 → panel_003 → ...
4. 배열 후 타임라인 확인

> **팁:** 파일명이 `panel_001`, `panel_002` 순으로 되어 있으면 정렬 후 한번에 추가 가능

---

### 4-3. 클립 길이 설정 (매우 중요)

스토리보드의 **권장 지속 시간** 기준으로 각 클립 길이를 설정합니다.

```
스토리보드 예시:
  panel_001 | Intro | wide       | 5초
  panel_002 | Intro | silhouette | 5초
  panel_003 | Verse | medium     | 4초
  panel_004 | Chorus| closeup    | 4초
```

**CapCut에서 클립 길이 조절:**
1. 타임라인에서 이미지 클립 탭
2. 클립 오른쪽 끝을 좌우로 드래그하여 길이 조절
3. 또는 클립 선택 후 하단 **편집** → **지속 시간** 직접 입력

> **기준:** 발라드(slow) 7~8초 / 팝(medium) 6초 / 빠른곡(fast) 4~5초

---

### 4-4. Ken Burns 효과 적용 (핵심)

정적인 이미지에 **느린 줌/패닝**을 넣어 생동감을 줍니다.  
웹툰 MV의 핵심 기법입니다.

**CapCut에서 Ken Burns 적용:**

1. 타임라인에서 이미지 클립 탭
2. 하단 메뉴 → **애니메이션**
3. **인 애니메이션** 탭에서 선택:
   - `줌 인` — 천천히 확대 (감성 패널에 적합)
   - `줌 아웃` — 천천히 축소 (와이드샷에 적합)
   - `왼쪽으로 이동` / `오른쪽으로 이동` — 패닝 효과
4. 속도: **느리게** (최대한 느리게 설정)
5. 전체 클립 길이에 걸쳐 적용

**패널 타입별 권장 효과:**

| 패널 타입 | 권장 Ken Burns | 이유 |
|----------|--------------|------|
| `wide` (와이드) | 줌 인 (느리게) | 세계관 서서히 드러남 |
| `closeup` (클로즈업) | 줌 인 (느리게) | 감정 고조 |
| `silhouette` (실루엣) | 줌 아웃 (느리게) | 여운, 분위기 |
| `medium` (미디엄) | 좌→우 패닝 | 퍼포먼스 느낌 |
| `detail` (디테일) | 줌 인 (느리게) | 악기 강조 |
| `crowd` (관객) | 줌 아웃 (느리게) | 규모감 표현 |
| `atmosphere` (분위기) | 좌→우 패닝 | 배경 묘사 |

---

### 4-5. 음악 파일 삽입

1. 타임라인 하단 → **오디오** → **음악 추가**
2. Suno에서 다운로드한 음악 파일 선택
3. 타임라인 시작점(0초)에 배치
4. 음악 길이와 이미지 전체 길이가 맞는지 확인

**이미지 총 시간 ≠ 음악 길이일 경우:**
- 이미지가 짧으면 → 마지막 클립 길이를 늘림
- 이미지가 길면 → 초과 클립 삭제 또는 음악 페이드 아웃 시점 조정

---

### 4-6. 클립 간 트랜지션

이미지 컷과 컷 사이에 전환 효과를 넣습니다.

1. 타임라인에서 클립 사이의 **|** 아이콘 탭
2. 추천 트랜지션:
   - **페이드 (Fade)** — 가장 자연스러움, 모든 상황에 적합
   - **크로스 페이드** — 감성적 분위기
   - **없음 (Cut)** — 에너지 있는 구간 (코러스 절정)
3. 지속 시간: **0.3~0.5초** (너무 길면 어색함)

**섹션별 트랜지션 전략:**

| 상황 | 트랜지션 |
|------|---------|
| Intro → Verse | 페이드 인 |
| Verse → Pre-Chorus | 크로스 페이드 |
| Pre-Chorus → Chorus | 빠른 컷 (무트랜지션) |
| Chorus → Verse | 페이드 |
| Bridge → Chorus | 빠른 컷 |
| 마지막 Outro | 페이드 아웃 |

---

### 4-7. 가사 자막 추가 (선택)

1. 타임라인에서 자막 넣을 시점으로 이동
2. 하단 메뉴 → **텍스트** → **텍스트 추가**
3. 가사 입력 (스토리보드의 가사 미리보기 참조)
4. 폰트: 굵고 읽기 쉬운 것 선택
5. 위치: 하단 1/3 영역
6. 해당 가사 구간에 맞게 클립 길이 조절

> **팁:** CapCut **자동 캡션** 기능을 사용하면 음악에서 자막을 자동 추출할 수 있음 (정확도 주의)

---

### 4-8. 추가 연출 (선택)

**색보정:**
- 하단 → **필터** → 영화적 필터 적용
- 또는 **조정** → 밝기/대비/채도 미세 조정
- 웹툰 스타일에는 **채도 약간 높임 + 대비 높임** 효과적

**오프닝 타이틀:**
- 첫 번째 클립 앞에 검정 배경 + 곡 제목 텍스트 삽입 (1~2초)

**엔딩:**
- 마지막 클립 뒤에 검정 페이드 아웃
- 채널명 또는 Suno 크레딧 텍스트 추가

---

### 4-9. 최종 내보내기

1. 우상단 **내보내기** (→ 또는 ↑ 버튼)
2. 설정:
   - 해상도: **1080p**
   - 프레임레이트: **30fps**
   - 품질: **권장** 또는 **높음**
3. 내보내기 완료 → 파일 저장

```
권장 저장 위치:
  image/[곡명]/[곡명]_MV_최종.mp4
```

## STEP 5 — 최종 확인 체크리스트

```
□ 이미지가 스토리보드 순서대로 배열되어 있는가?

□ Ken Burns 효과가 모든 클립에 적용되어 있는가?
□ 음악 시작과 영상 시작이 맞춰져 있는가?

□ 클립 사이 트랜지션이 자연스러운가?
□ 캐릭터(스켈레톤 밴드)가 영상 전체에서 일관된가?
□ 1080p로 내보냈는가?
```

## 참고 — 작업 효율 팁

**이미지 생성 순서:**
1. Chorus 패널 먼저 → 전체 톤 확인
2. Intro/Outro 패널 → 세계관 기준 확립
3. 나머지 Verse/Bridge 패널 순서대로

**CapCut 단축키 (PC 버전):**
- `Space` — 재생/정지
- `←` / `→` — 1프레임 이동
- `Ctrl + Z` — 실행 취소
- `Ctrl + S` — 저장

**재사용 팁:**
- 잘 나온 이미지 → 같은 섹션 반복 구간에 재사용 가능 (Chorus 1 = Chorus 2)
- 배경/분위기(atmosphere) 패널은 인트로·아웃트로에서 반복 사용 자연스러움
