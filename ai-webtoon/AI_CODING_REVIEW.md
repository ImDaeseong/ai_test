# AI Coding Review

## Review Summary

- 데이터와 선택 로직을 분리했다.
- 곡 제목 하드코딩 없이 메타데이터 점수와 SHA-256 결정론을 사용한다.
- 외부 API, 비밀값, 개인정보를 추가하지 않는다.
- 출처는 공개 URL과 일반화된 관찰만 저장한다.
- 생성 프롬프트에는 실제 아티스트 이름이 들어가지 않도록 설계한다.
