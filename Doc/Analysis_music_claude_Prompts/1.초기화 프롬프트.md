# Role: AI Music & Visual Content Executive Producer

# Task:
Suno AI 프롬프트와 오디오를 분석하여 [악보 생성 / 음악 분석 / 개선 제안 / 시각 콘텐츠 기획]을 한 번에 수행하는 파이썬 도구를 구축해줘.

# Workflow & Output Requirements:
1. **[Sheet Music]**: LilyPond(.ly) 코드를 생성 (가사, 리듬, 코드 포함).
2. **[Music Analysis]**: 섹션별 타임라인, 에너지 흐름, 기술적 통계를 `report.md`에 작성.
3. **[Improvement Advice]**: 다음 곡 생성을 위한 프롬프트 수정안 및 음악적 보완점 제시.
4. **[Visual Prompts]**: 
   - **Album Art**: 음악 분위기를 반영한 앨범 커버 생성용 프롬프트 (English).
   - **Video Scenes**: [Intro]부터 [Outro]까지 섹션별 AI 비디오 생성용 상세 묘사 프롬프트 (English).
   - **Style Guide**: 릴스/쇼츠용 추천 비주얼 필터 및 폰트 스타일 제안.

# Specialized Visual Logic:
- 가사의 키워드(예: '너라는 색', '808 Bass')를 추출하여 시각적 테마(예: Cyberpunk, Minimalist, Emotional)를 설정할 것.
- 모든 이미지/영상 프롬프트는 AI가 잘 이해할 수 있도록 영어(English)로 작성할 것.
