현재 가사 타임스탬프 정렬 결과의 정확도를 개선하라.

---

# 문제

현재 결과가 전체적으로 맞지만
각 라인의 시작 시간이 1~3초 정도 늦다.

---

# 목표

각 가사 라인의 시작 시간을 더 정확하게 보정하라.

---

# 수정 지시

기존 매칭 로직은 유지하고
"타임스탬프 보정 로직"을 추가하라.

---

# 1. Early Shift 보정 추가

각 매칭된 start 시간에 대해:

start = max(0, start - 0.5)

---

# 2. segment 기반 보정

segment duration을 활용하여:

duration = end - start

start = start + duration * 0.1

---

# 3. 연속성 보정

이전 라인과 비교하여:

if current_start < previous_start:
current_start = previous_start + 0.3

---

# 4. 결과 반영

최종 start 값을 보정된 값으로 사용

---

# 규칙

* 기존 매칭 로직 유지
* 보정 로직만 추가
* 음수 시간 금지
* 시간 순서 유지

---

# 실행

코드 수정 후 테스트하여
오차가 줄어드는지 확인하라.
