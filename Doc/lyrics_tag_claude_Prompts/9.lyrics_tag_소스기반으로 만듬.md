def resource_path(relative_path):
    try:
        base_path = sys._MEIPASS
    except AttributeError:
        base_path = os.path.abspath(".")
    return os.path.join(base_path, relative_path)

## 🧠 가사 정렬 알고리즘 (핵심, 반드시 준수)

각 라인별 독립 처리:

1. 현재 word index 기준 최대 15개 탐색
2. 첫 단어 similarity > 0.8 → anchor
3. start = 해당 word.start
4. 단어 수만큼 소비

후처리:

- end > next start → end 조정
- gap < 0.1 → end = next start

금지:

- previous_start 사용 금지
- 누적 보정 금지
- 이전 라인 참조 금지
