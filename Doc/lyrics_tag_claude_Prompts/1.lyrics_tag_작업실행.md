지금부터 즉시 작업을 실행하라.

현재 디렉토리(lyrics_tag)에서 바로 작업하라.
새 폴더를 만들지 마라.

설명하지 말고 파일 생성과 코드 작성을 수행하라.

---

# 1. 폴더 구조 생성

mkdir -p templates
mkdir -p static

---

# 2. 파일 생성

touch app.py
touch requirements.txt
touch templates/index.html
touch static/app.js
touch static/style.css

---

# 3. requirements.txt 작성

flask
flask-cors
openai-whisper
torch

---

# 4. 패키지 설치

pip install -r requirements.txt
pip install torch --index-url https://download.pytorch.org/whl/cpu || pip install torch

---

# 5. app.py 작성

다음 기능을 구현하라:

* Flask 서버 (포트 5000)
* CORS 허용
* Whisper base 모델

/api/transcribe:

* 파일 업로드 받기
* Whisper 전사
* 결과 반환:
  [
  { text, start, end }
  ]

/api/export_lrc:

* 입력:
  [
  { text, start }
  ]
* LRC 형식 생성
* 파일 다운로드

---

# 6. index.html 작성

* Drag & Drop 업로드 영역
* 파일 선택 버튼
* 오디오 플레이어
* 가사 리스트 영역
* LRC 다운로드 버튼

---

# 7. app.js 작성

* 파일 업로드 처리
* Whisper 요청
* 결과 렌더링
* 라인 선택
* 오디오 싱크 highlight

키보드 기능:

* Space → 현재 시간 입력
* Enter → 다음 줄
* P → 재생/정지
* ← / → → 2초 이동

---

# 8. style.css 작성

* 다크 테마
* active 라인 강조

---

# 9. 서버 실행

python app.py

---

# 10. 테스트

* http://localhost:5000 접속
* 파일 업로드
* 전사 확인
* Space 타이밍 입력 확인
* LRC 다운로드 확인

문제 발생 시 수정 후 재실행하라.
