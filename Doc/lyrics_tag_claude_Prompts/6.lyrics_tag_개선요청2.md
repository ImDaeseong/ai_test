# 수정 대상 파일

app.py

---

# 작업

1. 가사와 Whisper segment를 매칭하는 코드 부분을 찾아라.

2. 그 안에서 start 시간을 계산하는 부분을 찾아라.

---

# 문제 코드 제거

아래와 같은 "이전 값 기반 보정"이 있으면 삭제하라:

if current_start < previous_start:
current_start = previous_start + 0.3

또는 previous_start를 사용하는 모든 로직 제거

---

# 수정 코드 적용

각 라인의 start 시간은 다음처럼 계산하라:

adjusted_start = max(0, segment_start - 0.3)

---

# 중요한 조건

* 각 라인은 독립적으로 계산해야 한다
* 이전 라인의 시간값을 참조하지 마라
* 보정은 한 번만 적용하라
