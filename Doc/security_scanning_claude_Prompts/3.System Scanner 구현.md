Implement modules/system_scanner.py for DefenseScan targeting Windows 10+:

1. Open Port Scanner
   - Use psutil + socket to detect listening ports
   - Map port to process name and PID
   - Flag suspicious ports (common malware ports list)

2. Process Monitor
   - List all running processes with: name, PID, path, CPU, memory
   - Detect unsigned processes making network connections
   - Flag processes running from Temp/AppData paths
   - Detect process name spoofing (known system process in wrong path)

3. Network Connection Monitor
   - Active connections per process
   - Flag unknown/foreign IP connections
   - Detect processes connecting to multiple IPs simultaneously
   - Idle-state unexpected connections

4. Startup Programs
   - HKCU and HKLM registry run keys
   - Startup folder entries
   - Flag unsigned or suspicious path entries

5. Security Software Status
   - Windows Firewall status (all profiles: domain/private/public)
   - Windows Defender status via PowerShell Get-MpComputerStatus
   - Fallback if PowerShell fails

6. File Permission Checker
   - Check weak permissions on: C:\Windows\Temp, C:\Temp, user AppData
   - Flag world-writable system directories

Requirements:
- Detect Windows version (platform.version) and handle differences
- Graceful fallback if admin rights not available
- All subprocess calls with timeout and error handling
- Use psutil, subprocess, winreg modules
