# 🛠️ main.py 구현 지침

## main() 진입점 흐름 (이 순서 반드시 준수)

```
1. URL 입력: sys.argv[1] 우선, 없으면 input()으로 입력받기
2. try_ytdlp(url) 시도
   → 성공: 완료 메시지 출력 후 종료
   → 실패: 다음 단계로 진행
3. find_stream_with_playwright(url) 호출
   → 결과 없음: 실패 메시지 출력 후 종료
4. 수집된 스트림 분류
   → m3u8 있으면: get_best_m3u8() → download_with_ffmpeg()
   → m3u8 없고 mp4만 있으면: download_with_ffmpeg() 직접 호출
```

## 필수 함수 구조

1. **try_ytdlp(url: str) -> bool**
   * yt-dlp 사용, `format='bestvideo+bestaudio/best'`
   * `outtmpl='%(title)s.%(ext)s'` 로 현재 디렉토리에 저장
   * 성공 시 `True` 반환, 실패 시 예외를 외부로 던지지 않고 `False` 반환

2. **find_stream_with_playwright(url: str) -> list[dict]**
   * `sync_playwright` 사용, chromium headless 모드
   * 페이지 로딩 중 request / response 이벤트 감지
   * 아래 조건 해당 URL 수집:
     * URL에 `.m3u8` 또는 `.mp4` 포함
     * Content-Type: `application/vnd.apple.mpegurl` 또는 `video/mp4`
   * `User-Agent`, `Referer` 헤더 함께 반환
   * `url` 기준 중복 제거 필수
   * 반환 형식: `[{"url": "...", "type": "m3u8|mp4", "headers": {...}}]`
   * playwright 미설치 시 안내 메시지 출력 후 종료:
     ```
     ❌ playwright install chromium 을 먼저 실행하세요.
     ```

3. **get_best_m3u8(m3u8_url: str, headers: dict) -> str**
   * `requests` 사용 (headers 포함), timeout=15
   * 응답 본문에 `#EXT-X-STREAM-INF` 포함 시 master playlist로 판단
   * `BANDWIDTH` 값 기준 가장 높은 화질의 URI 선택
   * 상대 경로인 경우 `urljoin`으로 절대 경로 변환
   * master playlist가 아닌 경우 원본 URL 반환

4. **download_with_ffmpeg(url: str, headers: dict, output_name: str = "")**
   * `output_name` 미지정 시 `output_YYYYMMDD_HHMMSS.mp4` 자동 생성
   * `shutil.which("ffmpeg")` 로 설치 여부 사전 확인
   * ffmpeg 미설치 시 안내 후 종료:
     ```
     ❌ ffmpeg가 없습니다. 설치: https://ffmpeg.org/download.html
        Windows: winget install ffmpeg
     ```
   * m3u8 URL: `-bsf:a aac_adtstoasc` 옵션 포함하여 mp4 변환
   * mp4 URL: `-c copy` 로 직접 저장
   * `subprocess.run(cmd, shell=False)` 으로 안전하게 실행
   * 저장 완료 후 파일 존재 여부 확인 및 결과 출력

# 🔧 추가 구현 요구사항

* ffmpeg 설치 여부 사전 체크 (`shutil.which`)

* m3u8 우선 선택, 없을 경우 mp4 사용
* Windows subprocess 안전 실행: `shell=False`, 인자 리스트 형태 전달
* 최고 화질 자동 선택 및 URL 중복 제거
