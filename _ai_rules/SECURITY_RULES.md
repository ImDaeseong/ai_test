# Security Rules

`ai-workspace`는 집과 회사에서 함께 사용할 수 있는 개인 AI 작업 저장소다. 정보의 성격을 기준으로 저장 가능 여부를 판단한다.

## Never Store

- 회사 소스코드
- 회사 내부 문서
- 고객 정보
- 비밀번호
- API 키
- 토큰
- DB 접속 정보
- 서버 주소
- 내부망 정보
- 사내 URL
- 미공개 제품/정책/계약 정보
- 개인 주민등록번호, 결제 정보, 민감한 건강 정보

## Allowed

- 개인 AI 실험 기록
- 개인 학습 메모
- 일반화된 개발 패턴
- 공개 자료 기반 정리
- 직접 만든 프롬프트
- Suno/YouTube/개인 자동화 아이디어
- 회사 정보가 제거된 추상화된 회고

## Before Commit Or Push

```text
변경 파일 목록:
회사 정보 포함 여부:
비밀값 포함 여부:
개인 민감정보 포함 여부:
공개 저장소로 바뀌어도 안전한 수준인지:
추가 확인이 필요한 파일:
```

## Cost Boundary

유료 AI 도구를 쓰기 전에는 아래를 먼저 기록한다.

```text
도구 이름:
월 비용:
필요한 이유:
무료 대안:
첫 3개 영상 제작에 꼭 필요한가:
이번 달 비용 한도 안에 있는가:
결정: 보류 / 1회 테스트 / 결제
```
