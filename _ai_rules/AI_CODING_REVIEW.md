# AI 코딩 코드 리뷰 체크리스트

> 복사 위치: 프로젝트 루트 `/AI_CODING_REVIEW.md`
> 사용 시점: AI가 생성한 코드를 커밋/머지하기 전
> 소요 시간: 15~30분
> 원칙 (Uncle Bob): "AI가 만들었어도 내 이름으로 배포하면 내 책임이다"

---

## 이해 가능성 검사 (Simon Willison 원칙)

> 이 섹션을 통과하지 못하면 나머지는 의미 없다

- [ ] 이 코드가 하는 일을 **나는 한 문장으로 설명할 수 있다**
- [ ] 데이터 흐름을 추적할 수 있다 (입력 → 처리 → 저장 → 출력)
- [ ] AI가 사용한 외부 라이브러리의 공식 문서를 직접 확인했다
- [ ] AI가 선택한 패턴이 이 프로젝트 맥락에서 적절하다는 것을 이해한다

```
이 코드가 하는 일 (한 문장으로 직접 작성):
```

## 보안 검사 (Troy Hunt + Schneier 원칙)

### 비밀값 노출
- [ ] 하드코딩된 API 키, 토큰, 비밀번호가 없다
- [ ] 하드코딩된 DB 접속 정보, 서버 주소가 없다
- [ ] 로그 출력에 민감정보가 포함되지 않는다
- [ ] 에러 메시지에 내부 시스템 정보가 노출되지 않는다

```bash
# 직접 확인 명령어 (변경된 파일 기준)
grep -r "password\s*=" .
grep -r "api_key\s*=" .
grep -r "secret\s*=" .
grep -r "token\s*=" .
```

### 입력값 처리
- [ ] 외부 입력값(사용자 입력, API 파라미터, 파일 업로드)에 검증 로직이 있다
- [ ] SQL 쿼리는 파라미터 바인딩을 사용한다 (문자열 포맷팅 금지)
- [ ] HTML 출력 시 이스케이프 처리가 있다 (XSS 방어)

### 인증/인가
- [ ] 보호가 필요한 엔드포인트에 인증 체크가 있다
- [ ] 권한 체크가 우회될 수 있는 경로가 없다
- [ ] 세션/토큰 만료 처리가 있다

### STRIDE 위협 빠른 점검
```
S (위조)     - 이 코드에서 신원을 위조할 수 있는가?
T (변조)     - 이 코드에서 데이터를 변조할 수 있는가?
R (부인)     - 이 코드의 행위가 감사 로그에 남는가?
I (정보노출) - 이 코드에서 민감정보가 새어나올 수 있는가?
D (서비스거부)- 이 코드에 무한루프, 메모리 과소비 위험이 있는가?
E (권한상승) - 이 코드에서 권한을 높일 수 있는가?
```

## 유지보수 가능성 검사 (Martin Fowler + Uncle Bob 원칙)

- [ ] 하나의 함수/클래스가 하나의 역할만 한다 (단일 책임)
- [ ] 기존 코드 변경 없이 기능을 확장할 구조인가
- [ ] 변수/함수 이름이 목적을 설명한다 (주석 없이도 이해 가능)
- [ ] 중복 코드가 없다 (AI는 복사-붙여넣기를 자주 한다)
- [ ] 함수 길이가 화면 한 페이지를 넘지 않는다

## 운영 가능성 검사 (Charity Majors 원칙)

- [ ] 에러 발생 시 로그에 충분한 정보가 남는다
- [ ] 외부 API 호출 실패 시 적절한 처리가 있다 (타임아웃, 재시도, 폴백)
- [ ] 비용이 발생하는 API 호출에 제한이 있다 (무한 루프 방어)
- [ ] 이 코드가 장애를 일으킬 때 어떻게 알 수 있는지 안다

## 자주 나오는 AI 코딩 패턴 경고

> 아래 패턴이 보이면 즉시 재검토

| 패턴 | 위험 | 올바른 방식 |
|---|---|---|
| `query = "SELECT * FROM users WHERE id=" + id` | SQL 인젝션 | 파라미터 바인딩 사용 |
| `console.log("API Key:", apiKey)` | 로그 노출 | 민감값 로그 제외 |
| `while(true) { callAPI() }` | 무한 과금 | 루프 제한 + 종료 조건 |
| `catch(e) {}` | 에러 무시 | 에러 로깅 + 적절한 처리 |
| `eval(userInput)` | 코드 인젝션 | eval 사용 금지 |
| `http://` (프로덕션에서) | 평문 전송 | `https://` 강제 |
| `SELECT *` | 과잉 데이터 노출 | 필요한 컬럼만 명시 |
