# 2. 처리 로직

## 경우 1: 가사 없음

* 기존 Whisper 결과 그대로 반환

---

## 경우 2: 가사 있음 (핵심 구현)

### STEP 1: Whisper 실행

result = model.transcribe(audio_file_path)

segments = result["segments"]

각 segment:

* text
* start
* end

---

### STEP 2: 가사 전처리

* 줄 단위로 분리
* 공백 제거
* 소문자 변환

lyrics_lines = [...]

---

### STEP 3: 텍스트 유사도 함수 구현

Python에서 간단한 유사도 함수 구현:

* difflib.SequenceMatcher 사용

def similarity(a, b):
return SequenceMatcher(None, a, b).ratio()

---

### STEP 4: 핵심 알고리즘 (Sequential Matching)

다음 조건을 만족하는 매칭 구현:

* 가사는 순서 유지
* segment도 순서 유지
* 이전 매칭 이후 segment부터 탐색

알고리즘:

for 각 가사 라인:
현재 segment부터 끝까지 탐색
similarity 가장 높은 segment 선택
threshold (예: 0.4 이상)일 때만 매칭
매칭되면:
해당 segment의 start 시간 사용
다음 탐색 시작 위치 업데이트

---

### STEP 5: 결과 생성

output:

[
{ text: "가사 라인", start: 12.34 }
]
