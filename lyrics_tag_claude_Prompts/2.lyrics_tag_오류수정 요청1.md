# 문제 원인

현재 코드에서 OpenAI API를 사용하고 있다.

즉, 아래와 같은 코드가 존재한다:

* import openai
* openai.Audio.transcribe(...)
* openai.api_key 사용

이 방식은 API 키가 필요하며 현재 환경에서는 사용할 수 없다.

로컬에서 실행되는 whisper 라이브러리를 사용하도록 수정하라.
