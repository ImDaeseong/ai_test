## API: /api/transcribe

### 입력

* audio file (필수)
* lyrics_text (선택)

---

## 처리 로직

### ✅ 가사 없음

Whisper 결과 그대로 반환:

[
{ text, start, end }
]

---

### ✅ 가사 있음 (핵심 기능)

#### STEP 1: Whisper 실행

result = model.transcribe(audio_file_path)
segments = result["segments"]

---

#### STEP 2: 가사 전처리

* 줄 단위 분리
* trim
* 소문자 변환

---

#### STEP 3: 유사도 함수

```python
from difflib import SequenceMatcher

def similarity(a, b):
    return SequenceMatcher(None, a, b).ratio()
```

---

#### STEP 4: Sequential Matching

조건:

* 가사 순서 유지
* segment 순서 유지
* 이전 매칭 이후부터 탐색

로직:

* 각 가사 라인마다
* 현재 segment index부터 탐색
* similarity 최고값 선택
* threshold ≥ 0.4일 때만 매칭

---

#### STEP 5: 보정 로직 (중요)

* 너무 짧은 segment 무시
* 매칭 실패 시:
  → 이전 시간 + 보간값
* start 역전 방지:
  → max(prev_start, current_start)

---

#### STEP 6: 출력

[
{ text: "가사", start: 12.34 }
]

---

## API: /api/export_lrc

* 파일 다운로드
